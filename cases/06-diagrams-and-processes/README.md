# Диаграммы и процессы

## Кратко

Для сервисов онлайн-записи и доставки подготовлены DFD, диаграммы с дорожками, расширенные процессные диаграммы, диаграммы и таблицы состояний.

## Что демонстрирует

- моделирование потоков данных;
- распределение действий между ролями;
- описание жизненного цикла объектов;
- сопоставление диаграммы состояний и таблицы переходов.

## Основные артефакты

- `artifacts/BRS/` — комплект из пяти материалов для онлайн-записи;
- `artifacts/DLV/` — комплект из пяти материалов для доставки;
- файлы `ex00_*_dfd.png` — DFD;
- файлы `ex03_*_dst.png` и `ex04_*_tst.xlsx` — состояния и переходы.

## Как смотреть

Начать с DFD, затем открыть диаграмму состояний и соответствующую таблицу переходов.

## Статус

Учебный кейс. Диаграммы описывают проектные предположения.
