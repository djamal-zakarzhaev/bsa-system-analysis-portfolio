управление расписанием, бронирование услуги,
просмотр записей и отзывов.

- использование связей включения и расширения.

- `artifacts/ex00_BRS_UC.docx`, `ex02_BRS_UC.docx`, `ex04_BRS_UC.docx` — описания сценариев;
