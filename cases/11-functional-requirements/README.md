# Функциональные требования

## Кратко

Проработаны статусные модели, жизненные циклы,
CRUD-операции, детальные действия и права доступа
для объектов сервиса онлайн-записи.

## Что демонстрирует

- проектирование жизненных циклов;
- описание переходов между статусами;
- детализацию операций и проверок;
- формирование ролевой модели доступа.

## Основные артефакты

- `artifacts/Ex00.docx`–`Ex02.docx` — объекты и статусные модели;
- `artifacts/Ex03.docx`–`Ex05.docx` — операции над справочниками и слотами;
- `artifacts/Ex06.docx` — права доступа ролей;
- основной фокус — жизненный цикл, детализация операций и матрица доступа.

## Статус

Учебный кейс. Требования не реализованы в приложении.
