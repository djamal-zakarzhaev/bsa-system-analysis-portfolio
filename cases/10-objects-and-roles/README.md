# Объекты и роли

## Кратко

Последовательно описаны ключевые сущности сервиса онлайн-записи: персоны, клиенты, сотрудники, справочники, слоты, скидки и уведомления. Модель дополнена диаграммами классов.

## Что демонстрирует

- спецификацию сущностей и атрибутов;
- описание назначения объектов;
- моделирование связей и внешних ключей;
- развитие модели данных по мере уточнения требований.

## Основные артефакты

- `artifacts/Ex00/`–`Ex03/` — спецификации базовых сущностей;
- `artifacts/Ex04/BSA09_Ex04.docx` — первая диаграмма классов;
- `artifacts/Ex05/BSA09_Ex05.docx` — дополнительные сущности;
- `artifacts/Ex06/BSA09_Ex06.docx` — актуализированная диаграмма классов.

## Как смотреть

Начать со спецификаций, затем сравнить диаграммы из `Ex04` и `Ex06`.

## Статус

Учебный командный кейс. Авторство командных результатов не заявляется как единоличное.
