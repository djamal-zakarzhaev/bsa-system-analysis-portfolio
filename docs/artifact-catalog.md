# Artifact catalog

This catalog is a reviewer-oriented layer over the original School 21 folder structure. “Present” means that a corresponding submission artifact exists and can be opened; it does not imply a course grade or stakeholder approval.

## Case-study threads

- **BRS / BAR / HBS:** online appointment booking for a barbershop, including schedules, services, booking, notifications, reviews, roles, and interface flows.
- **DLV / DLVR / DMS:** delivery-order management, including stakeholders, operational problems, data flows, roles, user stories, and target-state models.

Abbreviations vary across the original assignments and are preserved in filenames.
