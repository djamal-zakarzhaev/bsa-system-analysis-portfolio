# Exercise 03 — Атрибуты производительности

## Подход к оценке

Так как в условиях задач не указано точное количество клиентов, курьеров, операторов, диспетчеров и администраторов, ниже приведены расчетные значения для первой очереди внедрения. Значения нужны для фиксации требований к производительности и могут уточняться после получения статистики эксплуатации.

Время отклика указано как целевое значение для 95% пользовательских запросов при штатной и пиковой нагрузке.
