## 3. Проверки и сообщения об ошибках при вводе некорректных данных

| Проверка | Когда срабатывает | Результат операции и сообщение |
| --- | --- | --- |
| Обязательное поле не заполнено | Пользователь сохраняет запись с пустым обязательным полем. | Запись не сохраняется. Сообщение: «Заполните обязательное поле». |
| Превышена допустимая длина | Значение текстового поля длиннее указанного ограничения. | Запись не сохраняется. Сообщение: «Превышена допустимая длина поля». |
| Неверный формат телефона | Телефон содержит недопустимые символы или слишком короткий номер. | Запись не сохраняется. Сообщение: «Неверный формат телефона». |
| Неуникальное значение | Код, телефон или другая уникальная запись уже существуют в соответствующем справочнике. | Запись не сохраняется. Сообщение: «Запись с таким значением уже существует». |
| Ссылка на неактивную запись | Пользователь выбирает неактивный филиал, роль, услугу, категорию, канал связи или сотрудника. | Запись не сохраняется. Сообщение: «Выберите активное значение из справочника». |
| Некорректное числовое значение | Цена меньше 0, превышает допустимый предел или содержит неверный формат. | Запись не сохраняется. Сообщение: «Проверьте числовое значение». |
| Некорректное время отправки | В правиле уведомления указано пустое или непонятное смещение времени. | Запись не сохраняется. Сообщение: «Укажите время отправки уведомления». |
| Некорректная связка мастер — услуга | В связке выбран неактивный мастер, неактивная услуга или неактивный филиал. | Запись не сохраняется. Сообщение: «Невозможно сохранить связку из-за неактивных связанных данных». |
| Дублирование связки мастер — услуга | Для одного мастера, услуги и филиала уже существует активная связка. | Запись не сохраняется. Сообщение: «Такая связка мастера и услуги уже существует». |
| Нет записей после поиска или фильтрации | По заданным параметрам список не содержит записей. | В списке отображается сообщение: «Записи не найдены. Измените параметры поиска или фильтра». |
