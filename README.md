# Портфолио системного и бизнес-анализа

Сводный репозиторий с аналитическими кейсами по требованиям,
стейкхолдерам, бизнес-процессам, моделям данных, интерфейсам
и управлению требованиями.

Портфолио подготовлено для публичного просмотра на GitHub
и ориентировано на позицию **Junior System Analyst**.

## О проекте

Репозиторий объединяет 14 учебных и практических кейсов
по системному и бизнес-анализу.

Внутри сохранены итоговые аналитические артефакты:
документы, таблицы, диаграммы, модели, отчёты и макеты интерфейсов.

Основные предметные области:

- сервис онлайн-записи в барбершоп;
- система управления доставкой заказов.

Материалы представлены как портфолио аналитических результатов,
а не как архив исходных учебных заданий.

## Что демонстрирует портфолио

- декомпозицию задач и ведение глоссариев;
- анализ заинтересованных сторон и их потребностей;
- описание текущего и целевого состояния процессов;
- применение методов выявления требований;
- моделирование данных, сущностей и связей;
- построение DFD, BPMN и диаграмм состояний;
- подготовку User Stories и Use Cases;
- описание объектов, ролей, операций и прав доступа;
- формулирование функциональных и нефункциональных требований;
- анализ пользовательских интерфейсов и экранных форм;
- регистрацию требований, трассировку и подготовку тест-кейсов.
